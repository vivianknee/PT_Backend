Java POJO Concepts Learned

  1. Encapsulation: bundling the data (variables) and the methods (functions) that work on the data into a single unit called a class.
  2. Inheritance: you can create a new class (child class) based on an existing class (parent class). The child class inherits the attributes and behaviors of the parent class.
  3. Abstraction: a way to define the essential functions or behaviors that users need to work with, while hiding the inner workings and complexities. Ex: private vs public variables
  4. Polymorphism: objects of different classes can be treated as objects of a common parent class. This allows you to write more flexible code because you can use the same method or interface to work with different types of objects.
